Breadth First Search

C Algorithm Graph Theory File I/O

Overview

A C implementation of the Breadth-First Search (BFS) algorithm developed for the Foundations of Algorithm module at the University of Colombo. The program reads a directed or undirected weighted graph from a text file, performs BFS traversal starting from the first node, computes the total edge weight, and identifies any nodes unreachable from the starting point.

Requirements Met

Reads graph data from a 'graph.txt' file.

Stores the graph using an adjacency matrix or adjacency list for efficient traversal.

Implements BFS traversal starting from the first node, outputting the full traversal path.

Computes the total weight of the graph, correctly accounting for directed vs undirected edge counting.

Detects and lists all unreachable nodes from the starting node, outputting "None" if all nodes are reachable.

Compiled and tested on Linux. The Makefile produces an executable named 'bfs' as per submission guidelines.

Input File Format

Line 1: Number of nodes (e.g. 5 )

Line 2: Node names separated by spaces (e.g. A B C D E )

Line 3: Graph type either 'directed' or 'undirected'

Remaining lines: Edges in the format node1 node2 weight (e.g. A B 5 )
