Portfolio Website

HTML CSS Responsive Personal Project

Overview

A fully responsive personal portfolio website built from scratch using only HTML and CSS, no frameworks or libraries. The site showcases my projects, skills, and background through a clean dark-themed design with smooth animations, a consistent color system using CSS variables, and a mobile-first responsive layout that works across all screen sizes.

Pages

Home

Hero section with introduction, social links, and a skills snapshot.

About

Personal info cards, bio paragraph, and a technical skills table.

Portfolio

Project cards grid linking to individual project detail pages.

Contact

Contact form with name, email, subject, and message fields.

Key Features

Fully responsive layout with breakpoints for desktop, tablet, and mobile using CSS media queries.

Consistent dark theme using CSS custom properties (variables) across all pages — no inline styles.

CSS-only animations including slide-down reveals, hover lifts, and smooth fill transitions on buttons and social icons.

Semantic HTML5 structure using header, main, section, article, aside, nav, and footer elements throughout.

Pure CSS hamburger menu for mobile navigation — no JavaScript required.
